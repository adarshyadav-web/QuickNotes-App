# QuickNotes App

A powerful, grid-based productivity tool for managing your daily tasks and thoughts. Built with LocalStorage persistence for a seamless user experience.

## 🚀 Features

- **CRUD Operations**: Create, Read, Update, and Delete notes.
- **LocalStorage Persistence**: Your notes are saved even after closing the browser.
- **Color Coding**: Organize notes with vibrant, distinct categories.
- **Responsive Grid**: Automatically adjusts to your screen size.

## 🛠️ Technologies Used

- **HTML5**, **CSS3**, **JavaScript**
- **LocalStorage API**
